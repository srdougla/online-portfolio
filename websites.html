CMS Data Viewer

For my final project in Database Systems, I created an interactive database viewer for data I pulled from the Center for Medicare & Medicaid Services (CMS). The user can select from 4 tables to view, query, update (edit/delete), and join -- all CRUD operations. I utilized Python (Flask APIs) and JavaScript to load data from CSV files into a database and write the functionality that converts form input in HTML to queries that actually run in the backend.

This website is not hosted publicly. Navigate through the gallery below to get an idea of how it works!
